🍽️ Zomato Data Analysis Project
📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on Zomato restaurant data using Python. The objective is to analyze customer preferences, restaurant ratings, votes, online/offline orders, and spending patterns.

🛠️ Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
📂 Dataset
The project uses a Zomato restaurant dataset containing information such as:

Restaurant Type
Ratings
Votes
Online Orders
Approximate Cost for Two People
📊 Analysis Performed
1. Restaurant Type Analysis
Identified the restaurant category preferred by most customers.
Result: Dining restaurants are the most preferred category.
2. Votes Analysis
Calculated total votes received by each restaurant type.
Result: Dining restaurants received the highest number of votes.
3. Rating Distribution
Analyzed restaurant ratings using histograms.
Result: Most restaurants have ratings between 3.5 and 4.0.
4. Average Spending Analysis
Studied the approximate cost for two people.
Result: Most couples prefer restaurants with an average cost of around ₹300.
5. Online vs Offline Ratings
Compared ratings for online and offline orders.
Result: Online orders generally receive higher ratings.
6. Offline Order Preference
Analyzed restaurant categories receiving more offline orders.
Result: Dining restaurants receive the majority of offline orders.
